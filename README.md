# HDPE Pipes Product Landing Page

A responsive product landing page built using **HTML, CSS, and JavaScript**, bundled with **Webpack** for development and production builds. The project demonstrates modern frontend practices including modular JavaScript, responsive layouts, dynamic UI components, and optimized asset bundling.

The application includes several interactive sections such as a product hero carousel, image zoom preview, feature highlights, dynamic FAQ accordion, testimonials slider, downloadable resources, and modal contact forms.

---

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Webpack
- Responsive Design
